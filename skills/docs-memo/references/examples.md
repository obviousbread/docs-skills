# Примеры служебок по типовым ситуациям

| Тема | Суть |
|------|------|
| Договор на КЭП | Прошу утвердить договор с [Название поставщика] для выпуска КЭП за [срок] вместо [текущий срок]. Стоимость: [сумма] руб./год. [N] сотрудников. |
| Пересылка почты | Прошу настроить автоматическую пересылку с [рабочий email] на [личный email] на период отсутствия на рабочем месте. |
| Залив архива | Уведомляю о заливе архивного помещения [дата], повреждено [N] единиц документов за [год]. Приложение: Акт о заливе № [N]. |
