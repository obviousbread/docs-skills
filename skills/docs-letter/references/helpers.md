# API generate.py - справочник

## Параметры страницы

- Формат: A4
- Поля: верх 2 / лево 3 / право 1.5 / низ 2 см
- Шрифт: Times New Roman 14pt (тело), 11pt (реквизиты в шапке), 8pt (футер)
- Межстрочный интервал: одинарный (1.0)
- Красная строка: 1.25 см
- Выравнивание: по ширине (тело), по центру (обращение, реквизиты)

## Импорт

```python
import sys
sys.path.insert(0, "skills/docs-letter")
from generate import create_letter
```

## Формат body_paragraphs

Каждый элемент - один абзац. Два формата:

**Строка** - обычный абзац с красной строкой:
```python
body_paragraphs=[
    "Первый абзац текста.",
    "Второй абзац текста.",
]
```

**dict** - абзац с дополнительными опциями:
```python
body_paragraphs=[
    "Обычный абзац.",
    {"text": "Заголовок раздела", "bold": True, "center": True, "indent": False},
    {"text": "Нумерованный пункт", "numbered": True},
    {"text": "Второй нумерованный", "numbered": True},
    {"text": "Буллет с тире", "dashed": True},
    {"text": "Абзац без красной строки", "indent": False},
]
```

Ключи dict:
- `text` (str) - текст абзаца
- `indent` (bool, default True) - красная строка 1.25 см
- `numbered` (bool) - нумерованный список Word (1., 2., 3., ...). Каждая группа подряд идущих `numbered` элементов стартует с 1.
- `dashed` (bool) - маркированный список Word с «–» (en-dash). Все группы используют один list-стиль; автопунктуация (`;` после каждого, `.` после последнего в группе)
- `bold` (bool) - жирный
- `center` (bool) - по центру

Списки реализованы через стандартный механизм Word (`numbering.xml`, `<w:numPr>`), а не как обычный текст с префиксом. Отступ маркера и текста: hanging 0.5 см (709 dxa).

## Формат адресата

Каждая строка - отдельный элемент списка `addressee_lines`:
```python
addressee_lines=[
    "Генеральному директору",
    "ФГБУ «Научный центр»",
    "",                        # пустая строка = визуальный разрыв
    "[И.О. Фамилия]",
]
```

## Исполнитель

Всегда размещается в теле документа после блока подписи, отделён 5 пустыми строками.

- Серый текст (#808080, 8pt), Times New Roman
- Без `executor_title`: строки «ФИО», «телефон»
- С `executor_title`: строки «ФИО,», «должность,», «телефон»

В колонтитул письма исполнитель не выносится.
