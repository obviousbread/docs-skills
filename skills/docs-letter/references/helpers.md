# API generate.py - справочник

## Параметры страницы

- Формат: A4
- Поля: верх 2 / лево 3 / право 1.5 / низ 2 см
- Шрифт: Times New Roman 14pt (тело), 11pt (реквизиты в шапке), 8pt (футер)
- Межстрочный интервал: одинарный (1.0)
- Красная строка: 1.25 см
- Выравнивание: по ширине (тело), по центру (обращение, реквизиты)

## Импорт

```python
import sys
sys.path.insert(0, "skills/docs-letter")
from generate import create_letter
```

## Параметры create_letter()

| Параметр | Тип | Описание |
|----------|-----|----------|
| `addressee_lines` | `list[str]` | Строки адресата. Пустая строка = визуальный разрыв |
| `greeting` | `str` | Обращение. По умолчанию «Уважаемые коллеги!» |
| `body_paragraphs` | `list[str \| dict]` | Абзацы тела (подробнее ниже) |
| `appendix_text` | `str` | Текст приложения (без слова «Приложение:»). Пусто = не добавлять |
| `signer_title` | `str` | Должность подписанта |
| `signer_name` | `str` | И.О. Фамилия подписанта |
| `doc_date` | `str` | Дата «ДД.ММ.ГГГГ». Пусто = подчеркивание |
| `doc_number` | `str` | Номер документа. Пусто = подчеркивание |
| `on_number` | `str` | «На №» для ответных писем. Пусто = стандартное подчеркивание |
| `executor_name` | `str` | ФИО исполнителя (Фамилия Имя Отчество) |
| `executor_title` | `str` | Должность исполнителя (опционально, для `executor_in_body=True`) |
| `executor_phone` | `str` | Телефон исполнителя |
| `executor_in_body` | `bool` | `True` = исполнитель в теле документа серым текстом (для многостраничных писем). По умолчанию `False` (колонтитул) |
| `output_path` | `str` | Путь сохранения |

## Формат body_paragraphs

Каждый элемент - один абзац. Два формата:

**Строка** - обычный абзац с красной строкой:
```python
body_paragraphs=[
    "Первый абзац текста.",
    "Второй абзац текста.",
]
```

**dict** - абзац с дополнительными опциями:
```python
body_paragraphs=[
    "Обычный абзац.",
    {"text": "Заголовок раздела", "bold": True, "center": True, "indent": False},
    {"text": "Нумерованный пункт", "numbered": True},
    {"text": "Второй нумерованный", "numbered": True},
    {"text": "Буллет с тире", "dashed": True},
    {"text": "Абзац без красной строки", "indent": False},
]
```

Ключи dict:
- `text` (str) - текст абзаца
- `indent` (bool, default True) - красная строка 1.25 см
- `numbered` (bool) - автонумерация: 1., 2., 3., ...
- `dashed` (bool) - маркированный список с «-», отступ первой строки 1.25 см, автопунктуация (`;` после каждого, `.` после последнего в группе)
- `bold` (bool) - жирный
- `center` (bool) - по центру

## Формат адресата

Каждая строка - отдельный элемент списка `addressee_lines`:
```python
addressee_lines=[
    "Генеральному директору",
    "ФГБУ «Научный центр»",
    "",                        # пустая строка = визуальный разрыв
    "[И.О. Фамилия]",
]
```

## Исполнитель: два режима

**Одностраничное письмо** (по умолчанию, `executor_in_body=False`):
- Колонтитул первой страницы, 8pt
- Формат: ФИО, телефон

**Многостраничное письмо** (`executor_in_body=True`):
- Обычный текст серым цветом (#808080, 8pt) после блока подписи
- Формат: ФИО, должность, телефон
- Требуется заполнить `executor_title`
