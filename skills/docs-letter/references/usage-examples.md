## 5. Письмо с расширенным блоком исполнителя

Если у исполнителя нужно вывести должность отдельной строкой, заполни `executor_title`. Без него блок состоит из ФИО и телефона.

Формат исполнителя с `executor_title`:
```
Фамилия Имя Отчество,
должность, ученая степень,
телефон
```

```python
create_letter(
    addressee_lines=[
        "Секретарю",
        "Генерального совета",
        "общественной организации",
        "",
        "[Фамилия И.О.]",
    ],
    greeting="Уважаемый [Имя Отчество]!",
    body_paragraphs=[
        "Обращаемся к Вам с просьбой...",
    ],
    signer_title="<leader_title из ~/.docs-plugin/org_details.md>",
    signer_name="<leader_name_nom из ~/.docs-plugin/org_details.md>",
    executor_name="[Фамилия Имя Отчество]",
    executor_title="начальник отдела стратегического развития",
    executor_phone="+7 (000) 000-00-00",
)
```
