# Подтип протокола: правила формирования заголовка

## Что такое подтип

Подтип протокола – свободная строка, которая отвечает на вопрос «протокол **чего?**»,
то есть стоит в родительном падеже. Именно эту строку агент передаёт аргументом
`subtype` в функцию `_make_title_block`.

## Как формируется заголовок

Функция `_make_title_block(doc, subtype)` добавляет два отдельных параграфа:

```
ПРОТОКОЛ
<подтип>
```

Оба параграфа – центрированные, жирные, 14 pt. Никакого разделителя между ними нет;
строки идут вплотную. Подтип вставляется дословно, без какой-либо автокоррекции падежа
или регистра, поэтому правильный падеж – ответственность агента, а не кода.

## Примеры корректных подтипов

| Подтип (передаётся в `subtype`) | Итоговый заголовок |
|---|---|
| `оперативного совещания` | ПРОТОКОЛ / оперативного совещания |
| `совещания у генерального директора` | ПРОТОКОЛ / совещания у генерального директора |
| `рабочей встречи по вопросам информатизации` | ПРОТОКОЛ / рабочей встречи по вопросам информатизации |
| `заседания комиссии по охране труда` | ПРОТОКОЛ / заседания комиссии по охране труда |
| `совещания руководителей структурных подразделений` | ПРОТОКОЛ / совещания руководителей структурных подразделений |

Все примеры ФИО в этом файле вымышлены.

## Типичные ошибки падежа

### Именительный вместо родительного

Самая частая ошибка: пользователь называет мероприятие в именительном падеже.

| Неверно (именительный) | Верно (родительный) |
|---|---|
| `оперативное совещание` | `оперативного совещания` |
| `рабочая встреча` | `рабочей встречи` |
| `заседание комиссии` | `заседания комиссии` |
| `совещание у директора` | `совещания у директора` |

### Лишние слова или неверный порядок

- «протокол оперативного совещания» – слово «протокол» не нужно: оно уже в первой строке заголовка.
- «оперативного совещания протокол» – нарушен порядок слов.

### Заглавные буквы без необходимости

Подтип пишется строчными буквами, кроме имён собственных.
Например: `совещания у генерального директора` – не `Совещания У Генерального Директора`.

## Как исправлять ошибки падежа

1. Задать мысленный вопрос: «протокол **чего?**» – ответ и есть нужная форма.
2. Если пользователь написал именительный падеж, преобразовать в родительный до передачи в `subtype`.
3. Если в строке уже есть слово «протокол», удалить его.

## Обязательное уточнение у пользователя

Подтип протокола **всегда нужно уточнять**, если пользователь его не указал явно.
Недостаточно знать, что создаётся «протокол совещания» – важно понять, какого именно.
Примерный вопрос агента:

> Уточните подтип протокола в родительном падеже – «протокол чего?»
> Например: «оперативного совещания», «рабочей встречи по вопросам закупок»,
> «совещания у заместителя директора по науке».
