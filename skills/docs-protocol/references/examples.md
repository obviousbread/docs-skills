# Эталонные текстовые блоки протокола совещания

Готовые образцы для сверки формата. Используются агентом как точка отсчёта при генерации блоков документа.

---

## Блок «Присутствовали»

Таблица без видимых границ. Три колонки: ФИО, разделитель «–», должность строчными буквами. Председатель в таблицу не включается – он выводится отдельной строкой выше. Порядок строк – по убыванию должности.

```
Присутствовали:

Бирюзов Б.Б.        –   заместитель директора по организационным вопросам
Васильков В.В.      –   начальник отдела кадров
Гранатов Г.Г.       –   главный специалист по информационным технологиям
Дымов Д.Д.          –   ведущий специалист отдела планирования
```

---

## Блок «РЕШИЛИ»

Нумерованные пункты. Каждый начинается с инфинитива без подлежащего. Строка «Ответственные:» выводится только при наличии ответственных. Строка «Срок:» выводится всегда. Подпункты – маркер «–» с отступом под основным пунктом.

```
РЕШИЛИ:

1. Утвердить план мероприятий по переходу структурных подразделений
   на электронный документооборот и ввести его в действие с 1 июля 2026 года.

   Ответственные: Бирюзов Б.Б., Гранатов Г.Г.
   Срок: 20.06.2026.

2. Обеспечить еженедельный мониторинг ключевых показателей эффективности
   подразделений и своевременное информирование руководства об отклонениях.

   Ответственные: Васильков В.В.
   Срок: постоянно.

   – Сводный отчёт направлять председателю не позднее каждого понедельника.
   – Данные за прошедшую неделю фиксировать в корпоративной системе учёта.

3. Рассмотреть предложения структурных подразделений по оптимизации штатного
   расписания и подготовить сводное заключение для представления на утверждение.

   Ответственные: Васильков В.В., Дымов Д.Д.
   Срок: 
```

Пункт 3: строка «Срок:» присутствует, но остаётся пустой – плейсхолдер для ручного вписывания даты после печати протокола. Это не ошибка.

---

## Блок подписи председателя

Слева – должность и краткое наименование организации. Справа – инициалы и фамилия. Должность строчными буквами, кроме первого слова, если это собственное наименование.

```
Директор ФГБУ «Научный центр»                           А.А. Алмазов
```

Формат «И.О. Фамилия» – инициалы предшествуют фамилии, пробел между ними обязателен. Точки внутри инициалов обязательны.
