# Стиль формулировок блока «РЕШИЛИ»

Руководство для агента, формирующего пункты решений в протоколе совещания.

---

## Основной пункт

Каждый пункт блока «РЕШИЛИ» – нумерованный абзац. Правила формулировки:

- Начинается с инфинитива глагола без подлежащего («Утвердить», «Направить», «Обеспечить», «Рассмотреть»).
- Заканчивается точкой.
- Пункт содержит только суть поручения; ФИО в текст пункта не включаются – они идут отдельно в строке «Ответственные».

## Строка «Ответственные»

Выводится только если в пункте передан список `responsible`. Формат:

```
Ответственные: Фамилия И.О., Фамилия И.О.
```

Технически: значения из списка объединяются через `«, »`, из результата снимается завершающая точка (если она уже была), затем добавляется ровно одна точка. Двойная точка не возникает.

Пример для двух ответственных: `Ответственные: Алмазов А.А., Бирюзов Б.Б.`

Если `responsible` не передан или пуст, строка «Ответственные» не выводится совсем.

## Строка «Срок»

Выводится всегда, независимо от наличия значения. Метка «Срок:» выделяется полужирным.

| Значение `deadline` | Результат в документе |
|---|---|
| `None` | `Срок: ` (пустой плейсхолдер для ручного вписывания) |
| `"постоянно"` | `Срок: постоянно.` |
| `"по запросу"` | `Срок: по запросу.` |
| `"01.06.2026"` (или любая дата DD.MM.YYYY) | `Срок: 01.06.2026.` |

Важно: `deadline=None` не означает ошибку и не заменяется текстом «не установлен». Это намеренный плейсхолдер для ситуаций, когда срок будет вписан от руки после печати протокола.

## Подпункты (subitems)

Подпункты выводятся маркером «–» (en-dash) с отступом под основным пунктом.

Текст подпунктов не нормализуется: агент передаёт формулировку пользователя без изменений. Допустимы как глагольная форма («Проверить исполнение до 15-го числа»), так и назывная («Отчёт об исполнении», «Контрольные показатели»).

---

## Примеры пунктов

### Пункт с двумя ответственными и датой срока

> 1\. Разработать план мероприятий по переходу на электронный документооборот и представить его на утверждение.
>
> **Ответственные:** Алмазов А.А., Велигин В.В.
>
> **Срок:** 15.06.2026.

### Пункт с постоянным сроком и подпунктами

> 2\. Обеспечить еженедельный мониторинг ключевых показателей эффективности подразделений.
>
> **Ответственные:** Бирюзов Б.Б.
>
> **Срок:** постоянно.
>
> – Сводный отчёт направлять руководителю не позднее понедельника.
>
> – Данные за прошедшую неделю фиксировать в корпоративной системе учёта.

### Пункт без ответственных и с пустым сроком (плейсхолдер)

> 3\. Рассмотреть предложения структурных подразделений по оптимизации штатного расписания и подготовить сводное заключение.
>
> **Срок:** 

Строки «Ответственные» в этом пункте нет, потому что список `responsible` не передан. Строка «Срок:» присутствует, но остаётся пустой – сотрудник, оформляющий протокол, впишет дату от руки.
