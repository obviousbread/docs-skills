# Примеры использования generate.py

## Заголовок временного скрипта

```python
import os, sys

runtime_dir = os.path.expanduser("~/.docs-plugin/runtime")
sys.path.insert(0, os.path.join(runtime_dir, "skills", "docs-protocol"))
from generate import create_protocol
```

---

## Пример 1. Оперативное совещание (минимальный набор)

Председатель, двое присутствовавших, два пункта РЕШИЛИ без ответственных.
`output_path=None` – имя файла выводится автоматически: `Протокол <subtype> <DD.MM.YYYY>.docx`
в каталог `output_dir_protocol` из `org_details.md` (или `~/` при его отсутствии).

```python
create_protocol(
    subtype="оперативного совещания",
    chair={
        "lastname": "Алмазов",
        "initials": "А.А.",
        "position": "Заместитель генерального директора по медицинской части",
    },
    attendees=[
        {
            "lastname": "Бирюзов",
            "initials": "Б.Б.",
            "position": "Начальник отдела организации медицинской помощи",
        },
        {
            "lastname": "Васильков",
            "initials": "В.В.",
            "position": "Главный специалист отдела качества",
        },
    ],
    items=[
        {
            "text": "Актуализировать перечень нормативных документов по профилю деятельности отдела.",
            "responsible": None,
            "deadline": None,
        },
        {
            "text": "Представить сводную информацию о выполнении плана на очередном совещании.",
            "responsible": None,
            "deadline": None,
        },
    ],
    output_path=None,  # имя формируется автоматически
)
```

## Пример 3. Рабочая встреча с подпунктами и сроком «постоянно»

`subitems` задаёт вложенные пункты (en-dash bullet), которые печатаются под основным пунктом.

```python
create_protocol(
    subtype="рабочей встречи",
    doc_date="02.06.2026",
    chair={
        "lastname": "Алмазов",
        "initials": "А.А.",
        "position": "Заместитель генерального директора по развитию",
    },
    attendees=[
        {
            "lastname": "Бирюзов",
            "initials": "Б.Б.",
            "position": "Руководитель проектного офиса",
        },
        {
            "lastname": "Васильков",
            "initials": "В.В.",
            "position": "Ведущий специалист отдела цифровой трансформации",
        },
    ],
    items=[
        {
            "text": "Внедрить систему мониторинга ключевых показателей проектной деятельности.",
            "responsible": ["Бирюзов Б.Б."],
            "deadline": "постоянно",
            "subitems": [
                "Разработать перечень показателей и методику их расчёта.",
                "Настроить выгрузку данных из информационных систем организации.",
                "Провести опытную эксплуатацию и доложить о результатах.",
            ],
        },
        {
            "text": (
                "Актуализировать шаблоны проектной документации "
                "с учётом действующих регламентов."
            ),
            "responsible": ["Васильков В.В."],
            "deadline": "30.06.2026",
        },
    ],
    output_path=None,
)
```
